REMINDER :- I had used this algorithm to create a pokedex (something out of the cartoon show called Pokemon). However, this code can be used as a common object detector or classifier if you use the right dataset.  

In this directory, make sure to make your own .model file to store the model after training it on your dataset. 
DATASET - Create your own dataset that will have more than 200 pictures for each category you want to detect to get accurate results.
